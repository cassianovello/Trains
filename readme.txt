Soluções baseadas em busca por profundidade para encontrar caminhos no mapa de Kiwiland.
O mapa deve ser fornecido através de um arquivo, e o caminho para o arquivo deve ser passado como parâmetro. Exemplo no arquivo "mapa.txt"

Para executar, exportar um "Runnable jar" e digitar no console:
java -jar trains.jar caminhodoarquivo
